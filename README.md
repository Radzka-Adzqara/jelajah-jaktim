# JELAJAH JAKTIM

Jelajah Jaktim merupakan aplikasi mobile berbasis Flutter yang dikembangkan sebagai media informasi wisata, budaya, dan kuliner khas Jakarta Timur. Aplikasi ini dirancang dengan tampilan sederhana, modern, dan mudah digunakan untuk membantu pengguna mengenal berbagai destinasi wisata serta budaya lokal Jakarta Timur.

## Deskripsi Aplikasi

Aplikasi ini menyediakan informasi mengenai tempat wisata, kuliner khas Betawi, budaya lokal, serta fitur quiz interaktif untuk meningkatkan pengetahuan pengguna mengenai Jakarta Timur.

Selain itu, aplikasi juga menyediakan fitur detail lokasi wisata yang dilengkapi dengan peta digital menggunakan OpenStreetMap sehingga pengguna dapat melihat lokasi tempat wisata secara langsung.

## Fitur Utama

- Login dan Register User
- Home Page (Beranda)
- Halaman Kategori Wisata
- Halaman Detail Wisata
- Halaman Favorit
- Halaman Profil Pengguna
- Quiz Edukasi Jakarta Timur
- Drawer Navigation
- Bottom Navigation Bar
- Detail Lokasi Menggunakan Maps
- Database SQLite

## Teknologi yang Digunakan

- Flutter
- Dart
- SQLite (sqflite package)
- OpenStreetMap
- Flutter Map
- URL Launcher

## Tampilan Aplikasi

### Login Page
Halaman awal untuk login pengguna.

### Home Page
Menampilkan rekomendasi wisata, kategori wisata, destinasi populer, serta fitur quiz.

### Kategori Page
Menampilkan daftar wisata berdasarkan kategori tertentu.

### Detail Page
Menampilkan informasi lengkap wisata meliputi:
- Nama tempat
- Lokasi
- Deskripsi
- Rating
- Harga tiket
- Jam operasional
- Peta lokasi

### Quiz Page
Fitur edukasi interaktif mengenai budaya, wisata, dan kuliner Jakarta Timur.

## Tujuan Aplikasi

Tujuan utama aplikasi ini adalah membantu masyarakat, pelajar, dan wisatawan lokal dalam memperoleh informasi mengenai Jakarta Timur secara lebih modern, praktis, dan interaktif.

## Cara Menjalankan Project

1. Clone repository:

```bash
git clone https://github.com/Radzka-Adzqara/jelajah-jaktim.git
```

2. Masuk ke folder project:

```bash
cd jelajah-jaktim
```

3. Install dependencies:

```bash
flutter pub get
```

4. Jalankan aplikasi:

```bash
flutter run
```
